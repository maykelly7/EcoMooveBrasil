# 🚀 Minha Moto Store

## 📋 Como usar:

1. **Substitua as imagens** na pasta `img/` pelas suas motos:
   - `moto1.jpg` (800x600px) e `moto1-thumb.jpg` (80x80px)
   - `moto2.jpg` e `moto2-thumb.jpg`
   - etc...

2. **Edite os dados das motos** no arquivo `js/main.js`:
   ```javascript
   nome: "SUA MOTO AQUI",
   preco: 45000,
   imagem: "img/sua-moto.jpg",
   thumb: "img/sua-moto-thumb.jpg"
